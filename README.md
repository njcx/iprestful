# IPrestful

用于IP地址查询，使用spring boot 开发而成，性能良好

    启动方式： java -jar  iprestful-0.0.1-SNAPSHOT.jar
    Linux下启动方式： nohup java -jar  iprestful-0.0.1-SNAPSHOT.jar  &



```bash

curl  "http://127.0.0.1:8090/get_ip?ip=114.114.114.114"

```
